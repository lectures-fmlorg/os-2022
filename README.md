## 講義の進め方

進捗状況に応じてブレイクアウトルームを選択し演習を行って下さい。

ルーム分け

- SSH する
- Apcche で HTML を公開する
- Apache で PHP を公開する
- MySQL にテーブルを作る
- PHP から MySQL に接続する
- 終わった人(最終課題相談)
- 終わった人(LAMP 環境で発展課題)

終わり次第、SA, TA に確認してもらい次のルームに進んでください。
